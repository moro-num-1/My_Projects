Floated Box

This is an example of how text wraps around a floated element. The turquoise box on the left uses float: left;, so the text flows around it, filling the space to the right and below. This technique is commonly used to wrap text around images or sidebars in web layouts. You can see that the text continues next to the floated box, instead of starting only below it.

If you add more text, it will keep wrapping around the floated element until there is no more space, then continue below. This is what is meant by "wrapping" in CSS.
